# Finora

Finora is a full-stack finance-focused web application designed to help users
learn, simulate, and analyze financial decision-making through interactive
modules and real-world inspired scenarios.

## Features
- Interactive financial simulations
- Time-travel market scenarios
- User progress tracking & leaderboards
- Modular backend architecture
- Modern frontend interface

## Tech Stack
### Frontend
- React
- JavaScript
- HTML/CSS

### Backend
- Node.js
- Express.js
- SQL
