# Finora

Finora is a full-stack financial learning and simulation platform designed to bridge the gap between theoretical finance concepts and real-world market behavior. The platform provides users with an interactive environment to explore financial decision-making, risk management, and market dynamics through realistic simulations and data-driven scenarios.

Unlike traditional finance learning tools, Finora emphasizes experiential learning by allowing users to actively participate in simulated markets, analyze outcomes, and understand the consequences of financial decisions under varying economic conditions.

## Features
- Interactive financial simulations
- Time-travel market scenarios
- User progress tracking & leaderboards
- Modular backend architecture
- Modern frontend interface
